## 📌 Featured Projects

### [Policy Seed](https://github.com/kotani-takumi0/policy-seed)
> 内閣官房ハッカソン データ活用賞受賞

過去の類似事業をEmbeddingsで検索し、推定予算を算出する政策シミュレーションツール。  
`FastAPI` `OpenAI Embeddings` `JWT認証` `ワークフロー管理`

### [Beeport](https://github.com/kotani-takumi0/beeport)
> AI支援開発の「過程」と「理解度」を可視化するポートフォリオプラットフォーム

MCPサーバーで開発ログを自動収集し、ログから4択クイズを生成してスキルスコアを算出。  
`React 19` `TypeScript` `FastAPI` `PostgreSQL` `Qdrant` `MCP`

### [豊島区2040年人口動態分析](https://github.com/kotani-takumi0/toshima-data-analysis)
> 学生データ分析 AWARD 2026 最優秀賞

豊島区の出生率予測・ワンルームマンション税の合成コントロール分析・23区定住率比較・場所愛着理論の統合分析。  
`Python` `因果推論` `合成コントロール法` `人口統計データ`

### [StratMind](https://github.com/kotani-takumi0/StratMind)
過去の意思決定事例からAIが「問い」を自動生成し、新規企画のセルフレビューを支援するツール。  
`FastAPI` `OpenAI API` `Embeddings` `コサイン類似度`

### [ジモカ](https://github.com/kotani-takumi0/jimoka)
> 関西ビギナーズハッカソン vol.7 出品

地元紹介カードアプリ。出身地の魅力をカード形式で共有する。  
`React` `TypeScript` `Vite` `Playwright（E2Eテスト）`

### [FirstByte](https://github.com/kotani-takumi0/firstbyte)
